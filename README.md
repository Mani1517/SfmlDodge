# SfmlDodge
Source code of My Game called Dodge made with SFML.


# Windows

Step: 1 -> Clone the repo.  
Step: 2 -> "make window" in Cmd.  
Step: 3 -> Get the Executable in bin/win32

# Linux

Step: 1 -> Clone the repo.  
Step: 2 -> "make linux" in Cmd.  
Step: 3 -> Get the Executable in bin/linux
